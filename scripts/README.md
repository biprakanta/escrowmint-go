# Scripts

This directory contains Redis Lua references for the EscrowMint mutation contract.

The Go package currently embeds the active Lua sources directly in [escrowmint/lua.go](/Users/biprakantapal/Desktop/codex-plugins/escrowmint-go/escrowmint/lua.go).

The on-disk scripts directory is currently partial and documentation-oriented, not the runtime source of truth.

These scripts are intended as the shared behavioral core for both Python and Go clients.
